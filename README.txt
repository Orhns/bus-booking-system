


Projemde iki adet database kullandım. 

Biri kesilen bilet bilgilerini tutmak diğeri ise sisteme kaydedilen (üye olan) youlcuların kişisel verilerini tutmak için.

Kullanıcı bilet bilgilerini seçtikten sonra alınacak ücreti (ücret tarifesine göre) belirleyip forma girmelidir. 
Ücret tarifesi koltuk seçimi alanının sağında bulunmaktadır.

Yolcunun üyeliği var ise bilet kesilmeden önce Üye kayıt işlemleri'nden sorgulama yapılmalıdır ve ücret ona göre belirlenmelidir.

Yolcunun üyeliği var ise veya yeni üyelik oluşturmak istiyorsa kullanıcı "Üye kayıt işlemleri" butonuna tıklayarak
gelen pencerede önce "Veri tabanına bağlan" butonuna tıklayarak bağlantıyı kurmalı, ardından yolcunun cep telefonu 
numarası ile üyelik bilgilerini sorgulayabilir ya da istenilen diğer işlemleri gerçekleştirebilir. Üye kayıt işlemi tamamlandıktan
sonra güvenlik için "Bağlantıyı kes" butonuna basarak kişisel verilerin bulunduğu veri tabanı bağlantısını kesmelidir.
